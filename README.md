# Java JDBC CRUD Project
This project is developed using Java JDBC and mysql.

## 
This project demonstrates database connecivity using JDBC.It performs CRUD Operation (Create, Read,Update,Delete) on a mysql  database.

 ## Technologies used
 -Java
 -JDBC
 -MYSQL
 -Eclipse IDE
 
## Featuers 
-Insert data 
-Display records
-Update records
-Delete records

## Databases
MYSQL is used to store and manage data
